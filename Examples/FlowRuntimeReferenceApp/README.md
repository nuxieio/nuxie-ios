# Nuxie Flow Runtime Reference App

Small native iOS app for manually loading the published flow runtime fixtures:

- `layout-paint`
- `published-font`
- `pressable-interaction`

Run it from the generated `NuxieSDK.xcodeproj` with the
`NuxieFlowRuntimeReferenceApp` scheme, or use:

```sh
make build-reference-app
make install-reference-app
```
